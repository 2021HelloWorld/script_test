2. 遥操命令

    1. 遥操设备环境运行

        1. Version pro

            1. 启动 CloudXR Runtime container

            ```Bash
            docker run -it --rm --name cloudxr-runtime \
                --user $(id -u):$(id -g) \
                --gpus=all \
                -e "ACCEPT_EULA=Y" \
                --mount type=bind,src=$(pwd)/isaac_workspace/openxr,dst=/openxr \
                -p 48010:48010 \
                -p 47998:47998/udp \
                -p 47999:47999/udp \
                -p 48000:48000/udp \
                -p 48005:48005/udp \
                -p 48008:48008/udp \
                -p 48012:48012/udp \
                nvcr.io/nvidia/cloudxr-runtime:5.0.1
            ```

    2. 场景相关

        1. 抓三个水果

            1. （非docker环境）

                ```Plain Text
                #遥操
                ##vision_pro
                python source/easim/cli/run_unified.py   --task pick_fruits_skill \
                  --mode teleop_record \
                  --teleop_device handtracking \
                  --enable_pinocchio \
                  --pick-fruits-teleop-tabletop-sync-mode hard_reset \
                  --dataset_file datasets/imit_learning/pick_fruit_kinematicC_test_260609_2051.hdf5
                
                ##pico
                python source/easim/cli/run_unified.py   --task pick_fruits_skill \
                  --mode teleop_record \
                  --teleop_device pico_handtracking \
                  --enable_pinocchio \
                  --pick-fruits-teleop-tabletop-sync-mode hard_reset \
                  --dataset_file datasets/imit_learning/pick_fruit_kinematicC_test_260609_2051.hdf5
                  
                #回放
                python source/easim/model_zoo/IL_workflow_office/run_0_R_replay_demonstration.py \
                  --task Easim-PickFruitsOffice-G1-O6-Bimanual-Replay-v0 \
                  --replay_mode kinematic \
                  --num_envs 1 --headless \
                  --replay_camera_mode stable \
                  --dataset_file datasets/imit_learning/pick_fruit_kinematicC_test_260609_2051.hdf5
                  
                #剔除人工检查后的hdf5异常数据，搬迁异常视频
                python source/easim/model_zoo/tools/remove_bad_replay_trials.py \
                  --dataset_file datasets/imit_learning/pick_fruit_kinematicC_test_260609_2051.hdf5 \
                  --remove_trials trial_1 trial_2 \
                  --video_dir outputs/videos/replay_demo/pick_fruit_kinematicC_test_260609_2051_20260609_130031 \
                  --move_failed_videos
                ```
